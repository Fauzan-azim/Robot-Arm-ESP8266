# Robot-Arm-ESP8266
A Robot Arm with ability to lift object and move around
  - Software 
    + Arduino IDE
    + Controll App 
    
  - Hardware and Software 
    + NodeMCU ESP8266
    + L289N Motor Driver
    + Ultrasonic Sensor HC-SR04
    + 4 DC Motor + Wheels
    + 2 Battery (13.000 mAh)
    + Acrylic Robot Arm
    + 2 Servo Motor
